# 暴騰ダンプ・スクリーナー v3 — 使い方説明書

## 1. このBotは何をするのか

OKXの全USDT建て先物（約336銘柄）を **15分ごとに自動スキャン**し、
「**急騰したあと暴落する確率が高い銘柄**」を見つけて Discord に通知するシステムです。

裁量トレードの**判断材料を最大化する**のが目的で、自動売買ではありません。
**最終的なエントリー判断・損切・利確はあなた自身が行います。**

## 2. 根本ロジック — なぜ「暴騰後ショート」なのか

### 戦略の核
> 「**過去に何度も急騰→暴落を繰り返している銘柄は、また同じ動きをする確率が高い**」

この再現性こそが唯一の優位性 (エッジ)。
個別の天井予測は不可能だが、「過去 56回急騰のうち 46回ダンプ (82%) してきた銘柄」が
今また急騰しているなら、**同じパターンが繰り返される蓋然性が高い**。

### Botのスクリーニング3段階

```
Stage 1: ふるい
  └ 24時間出来高 ≥ $2M（流動性確保）
  └ 24時間 ±5% 以上動いている

Stage 2: 暴騰判定
  └ 1時間 +10% 以上、または 24時間 +20% 以上

Stage 3: 深堀り分析（候補ごと）
  ├ 過去300日のチャートから「±15%急騰」イベントを全部抽出
  ├ その後 3日以内に半値以上戻したかをカウント
  ├ FR / OI / L/S / CVD / ボラ / BTC相関 を取得
  └ 4hトレンド状態を判定
```

## 3. 通知の読み方

### 例: 🟡 監視: NIGHTUSDT [スコア 4/10 | 再現性 高]

#### ① タイトル行
| 表示 | 意味 |
|---|---|
| 🔴 強推奨 (スコア 7-10) | 全シグナル一致。条件揃えば即エントリー候補 |
| 🟡 監視 (スコア 4-6) | 部分一致。チャート確認してから判断 |
| ⚪ 弱 (スコア 0-3) | 参考情報程度 |

| 再現性 | 意味 |
|---|---|
| 極高 | 過去15回以上 × 80%以上ダンプ |
| 高 | 過去8回以上 × 70%以上ダンプ |
| 中 | 過去4回以上 × 60%以上ダンプ |
| 低 | データ不足または再現性なし |

#### ② 値動き
現在の価格・1時間/24時間の変化率。

#### ③ シグナル根拠
| 指標 | 何を見るか |
|---|---|
| **FR** (Funding Rate) | +0.05%超 = ロング過熱、ショートでFR受取可 |
| **OI変化** | +5%超 = ポジション急増 = 過熱サイン |
| **L/S比** | 1.5超 = 個人ロング偏重 = 逆張りチャンス |
| **1h売買差 (CVD)** | マイナス = 売り優勢 |
| **24h出来高** | 流動性の指標 |
| **ボラ** | 高いほど値動き激しい |
| **BTC相関** | 低いほど独自材料で動いている |
| **カテゴリ** | MEME/GameFi/AI は再現性が高い傾向 |

#### ④ 過去パターン
**最重要セクション**。例：
> 過去15%超の急騰 11回 → 10回ダンプ (91%)
> 平均戻し幅 119% / 約2.0日 → ほぼ全戻し傾向

これは「**この銘柄は過去ほぼ全部、急騰したら2日以内に元の位置まで戻ってきた**」という意味。
今もその確率で同じことが起きると想定する。

#### ⑤ トレンド / エントリープラン
- **4時間足の状態**: まだ上昇中か / 失速中か / 下落トレンド入りか
- **エントリートリガー**: 「4h終値が $X.XX を下抜けたら入る」
- **損切**: 当日高値 + 2%（リバ食らったら即撤退）
- **利確1**: 半値戻し（最低限ここまでは取る）
- **利確2**: 全戻し（再現性が高ければここまで狙う）

#### ⑥ チャート画像
- 72時間分のローソク足 + 出来高
- 赤線: 損切ライン (SL)
- 黄線: エントリートリガー (Trig)
- 緑線: 利確1 (TP1)
- 青線: 利確2 (TP2)

## 4. 推奨ワークフロー（通知が来てから）

```
Discord通知が来た
   ↓
①「スコア」と「再現性」を見る
   スコア7以上 & 再現性 高〜極高 = 本気で見る価値あり
   それ以下 = スルー or 軽くチャート確認のみ
   ↓
②「過去パターン」を読む
   ダンプ率80%以上、平均戻し幅50%以上 → 再現性◎
   ↓
③ 添付チャートで現状確認
   どこまで上がっているか / 直近の動きはどうか
   ↓
④ CoinGlassリンクで深堀り（任意）
   清算ヒートマップで「上に大きな清算クラスター」があるか
   そこで暴騰勢が清算されて急落する可能性
   ↓
⑤ 4時間足の状態判定
   「まだ上昇中」 → ブレイクダウン待ち（焦るな）
   「失速中・下落トレンド入り」 → エントリー候補
   ↓
⑥ エントリー実行
   通知の「エントリートリガー価格」を下抜けたらショート
   ロット・レバは自分で調整（ミーム系はボラ大なので注意）
   ↓
⑦ 損切・利確を即時セット
   SL: 通知の「損切」価格 (当日高値+2%)
   TP1: 「利確1」価格 (半値戻し)
   TP2: 「利確2」価格 (全戻し)
   ↓
⑧ リバったら即切る（資金管理が命）
```

## 6. 戦略上の重要原則

### A. 「天井はわからない」を前提にする
- 急騰中の銘柄は +30% から +50% にさらに伸びることもある
- だから**ブレイクダウン確認後**に入る（順張りショート）
- 上昇中の「逆張りショート」は禁止

### B. 損切は最優先
- リバ（買い戻し）が来たら即撤退
- 「もう少し待てば下がる」は破滅への道
- 通知の損切ライン（当日高値+2%）は必ず守る

### C. 再現性が全て
- 過去ダンプ率が低い銘柄は無視
- 「今回は違うかも」は通用しない
- 統計的有意性のある銘柄だけ取る

### D. 利確は段階的に
- TP1（半値戻し）で**最低50%**は利確
- 残りはTP2（全戻し）まで持つ、または建値SLに動かす
- 「全部TP2まで持つ」は欲張りすぎ

### E. FR受取は副次的なボーナス
- FR +0.05%以上の銘柄をショートすると、保有中にFRを受け取れる
- 1日 0.05%×3回 = 0.15% / 365日 = 約55%/年のキャリー
- ただし**価格変動の損益が主**、FRはおまけ

## 7. 激熱条件（コメントから抽出）

以下に該当する銘柄は特に再現性が高い傾向：

- ✅ **ミームコイン** (DOGE, PEPE, WIF, BONK, etc.)
  - 信者が多く、感情で動く → チャートに大衆心理が表れる
- ✅ **GameFi/BCG** (GALA, AXS, IMX, etc.)
  - 期待先行で上がり、現実で売られる
- ✅ **AI関連** (FET, WLD, AIXBT, etc.)
  - 上も下も激しい → ロットは少なめに
- ✅ **時価総額500〜1000位**
  - 流動性ありつつ、操作されやすいゾーン

逆に **BTC/ETH/SOL/BNB/XRP** は除外済み（メジャーすぎて再現性低い）。

## 8. 通知が来ないときの解釈

- 無理にエントリーしない時間も大事
- 1日数回〜0回の日もある（暴騰がなければゼロ）

## 11. トラブルシューティング

| 症状 | 原因 / 対処 |
|---|---|
| 通知が全く来ない | 相場が静か（正常）。GitHub Actions の実行履歴を確認 |
| Workflow が Failure | Actions タブでログ確認。OKX API障害なら時間が解決 |
| Webhook URL を変えたい | GitHub Settings → Secrets → `DISCORD_WEBHOOK_URL` を更新 |
| 通知が多すぎる | `PUMP_1H` を 15-20% に上げる |
| 通知が少なすぎる | `PUMP_1H` を 7-8% に下げる（ノイズ増加注意） |

## 12. 最後に — このBotの限界

- **過去のパターンが未来を保証するわけではない**
- 暗号資産は常に大暴騰・大暴落のリスクあり
- このBotは「**確率を上げる道具**」であり「**勝てる必勝法**」ではない
- 必ず**余剰資金**で、**少額から**始めること
- 損切ラインを守れば、長期的には期待値プラスになる戦略

> **Botは判断材料を提供する。トレードするのはあなた。**
> **資金管理が命。負けないために入る。**
